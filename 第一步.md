产品名：
miniAtoms

主标题：
Build apps with AI Agents

产品定位：
一个 Lovable 风格的 AI Agent App Builder。
用户通过自然语言描述应用想法，系统调用硅基流动 API 生成结构化应用方案，并将其渲染为可视化应用预览。用户需要登录后才能生成、保存和管理项目。

核心技术栈：
Next.js + React + TypeScript + Tailwind CSS + lucide-react + Clerk + Neon + SiliconFlow API

登录系统：
使用 Clerk 实现注册、登录、弹窗登录、用户状态和用户头像。

数据库：
使用 Neon PostgreSQL 保存项目数据。
每条项目记录绑定 Clerk user_id。

首页交互：
用户在首页 Prompt 框点击发送。
如果未登录，弹出登录 / 注册窗口。
登录成功后，继续进入 Builder 并生成刚才输入的应用需求。

受保护页面：
/builder
/projects
/projects/[id]

核心页面：
/
Lovable 风格首页

/builder
AI Agent 生成工作台

/projects
当前用户的历史项目

/projects/[id]
项目详情与继续迭代
